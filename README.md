# Syncode ⚡

Syncode is a real-time collaborative coding platform that allows multiple users to write and edit code together simultaneously. It enables seamless collaboration through shared coding sessions with live synchronization.

## ✨ Features

- 👥 Real-time collaborative code editing
- ⚡ Instant synchronization between users
- 🔗 Shareable rooms for collaboration
- 🔐 User Authentication (signup/login)
- 🧑‍💻 Multi-user coding environment
- 📝 Clean and responsive code editor UI

## 🛠️ Tech Stack

- **Frontend:** React 
- **Build Tool:** Vite
- **Authentication:** Jwt
- **Realtime Communication:** WebSockets / Socket.io 
- **Styling:** Tailwind CSS

## 🚀 Use Cases

- Pair programming
- Technical interviews
- Teaching / live coding sessions
- Team collaboration

## 🎯 Key Highlight

Implemented secure user authentication along with real-time multi-user collaboration, ensuring a smooth and interactive coding experience.
